# Angular Concepts

A simple Angular starter project that demonstrates common Angular concepts such as components, data binding, directives, services, routing, forms, and HTTP.

## Features

- Modern Angular standalone app structure
- Responsive landing page showcasing core concepts
- Ready for GitHub deployment and further learning

## Run locally

1. Install dependencies
   - `npm install`
2. Start the development server
   - `npm start`
3. Open your browser at
   - `http://localhost:4200/`

## Build for production

- `npm run build`

## GitHub upload

Initialize git in the project folder and push to your GitHub repository:

```bash
git init
git add .
git commit -m "Initial Angular concepts project"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```
