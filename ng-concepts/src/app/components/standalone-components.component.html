Standalone components

Use Angular without NgModule

Standalone components let you build Angular features with less boilerplate and a simpler composition model.

Standalone Components

Standalone components can be declared without an NgModule and can import other standalone pieces directly.

Standalone Pipes and Directives

Pipes and directives can also be standalone, making them easier to reuse across features.

Bootstrap Application

Standalone apps are bootstrapped directly with bootstrapApplication, which simplifies startup.
